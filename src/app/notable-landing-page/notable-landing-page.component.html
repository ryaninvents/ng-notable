ng-notable

Demo application written in Angular 6

View on Github

This was just a simple project so I could finally learn AngularJS. I didn't think I'd like it, since my only exposure to Angular was back in 1.0. I'm not going to lie, I still prefer React, but Angular's pretty nice too!

Create notes with the button at bottom-left. Notes are stored to your browser's localStorage, so they'll be deleted if you clear out your browser.
